
# ⚙️ AI Bazaar – Backend

Backend service for AI Bazaar providing authentication, API endpoints, and data management for shopkeeper operations.

## 📌 Features
- User authentication (JWT-based login)
- RESTful APIs for frontend integration
- Database integration 
- Secure data handling
- CORS-enabled for frontend communication

## 🛠️ Tech Stack
- Python
- FastAPI / Flask
- Supabase (PostgreSQL)
- REST APIs
